# 🌐 NM Project — Soil Nutrients Monitoring & Management

A premium, interactive web interface presenting soil nutrients monitoring & management phase documents, presentation materials, and project team information.

## ✨ Features
- **Dynamic Particle Canvas**: Custom wave-like canvas animations in the background (`interactive-bg.js`).
- **File Downloads**: Direct download triggers for project presentation (`NM_Presentation.pptx`) and project documentation (`NM_Final_Doc.pdf`).
- **Interactive Navigation**: Clean transitions between main info, phase documentation views, and team info.
- **Sleek Back Navigation**: Integrated back buttons with modern slide hover transitions (`transform: translateX(-4px)`) and intelligent browser history referrers.
- **Glassmorphic Theme**: Sophisticated dark-mode aesthetics using premium Satoshi and Inter font face configurations.

## 🛠️ Technologies
- **Core**: HTML5, Vanilla JavaScript (ES6+), Canvas API
- **Styling**: Vanilla CSS3 Custom Properties (CSS variables), CSS backdrop-filter, flexbox, CSS grid
- **Fonts**: Satoshi (via Fontshare), Inter & JetBrains Mono (via Google Fonts)
